## Descrição

O aplicativo realiza cálculos para um circuito com transistor NPN na configuração emissor comum, retornando valores importantes para a análise do circuito, como:

- Corrente de base (Ib)  
- Corrente do coletor (Ic)  
- Corrente da fonte  
- Potência dissipada  
- Outros parâmetros elétricos relevantes  

O objetivo é facilitar a verificação rápida de resultados e auxiliar no estudo do funcionamento do transistor nessa configuração.
